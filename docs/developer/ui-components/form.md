## Supported Properties

EarnKeeper does not support all properties of the underlying control, the currently supported properties are below.

| Name     | Type           | Description |
| -------- | -------------- | ----------- |
| name     | Rpc \| string  |             |
| schema   | Record<string, any> |             |
| child    | UiElement      |             |
