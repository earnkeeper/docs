# Table

Display a static table. Rendered as an HTML `<table>` element.

## Supported Properties

| Name    | Type                         | Description                    |
| ------- | ---------------------------- | ------------------------------ |
| header? | string\[] \| Rpc             | The header cells for the table |
| body    | string\[] \| Rpc             | Body cells for the table       |
